Frequently Asked Questions

Is MediCare Cloud free to use?

Yes! MediCare Cloud is hosted on Azure Free Tier which provides free hosting for static web apps and limited backend usage.

Where is my health data stored?

All structured data (medicines, reminders) is stored in Azure SQL Database. Health record files (PDFs, images) are stored in Azure Blob Storage.

How do email reminders work?

The Python Flask backend on Azure App Service runs a scheduler that sends emails at your set reminder times using Azure Communication Services.

Can I access my data from any device?

Yes! Since everything is stored in the Azure cloud, you can access MediCare Cloud from any mobile, tablet, or desktop browser.
