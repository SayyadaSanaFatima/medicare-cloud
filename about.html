About MediCare Cloud

A cloud-powered healthcare management system built for the SITER Academy Summer Internship 2026, Cloud Computing domain, using Microsoft Azure.

Our Mission

MediCare Cloud was created to solve a real problem — patients, especially elderly individuals managing multiple medications, often forget to take medicines on time. Health records like prescriptions and lab reports are frequently lost. MediCare Cloud solves both problems using Microsoft Azure cloud services.

Cloud Architecture

Three-tier architecture fully hosted on Microsoft Azure

🌐

Frontend Layer

HTML, CSS, JavaScript pages hosted on Azure Static Web Apps. Fully responsive and SEO-optimized.

⚙️

Backend Layer

Python Flask REST API hosted on Azure App Service. Handles authentication, medicine data, and file operations.

🗄️

Data Layer

Azure SQL Database for structured data. Azure Blob Storage for health record files (PDFs and images).
